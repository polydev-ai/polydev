## Visual Style Guide

**Color Palette:**
- Primary: Slate (#0f172a, #1e293b)
- Accent: Amber (#f59e0b) for highlights
- Background: White (#ffffff) or subtle gradient

**Typography:**
- Headlines: Inter Bold, 48-72px
- Body: Inter Regular, 24-32px
- Code: JetBrains Mono

**Animation Style:**
- Smooth, professional transitions
- Spring physics for UI elements
- Fade and slide for text reveals

**Model Logos:**
- OpenAI (green)
- Anthropic (coral/orange)
- Google (blue/Gemini colors)
- xAI/Grok (black/white)

**IDE Logos:**
- Claude Code
- Cursor
- Cline
- Windsurf
- Continue
