# **OpenAI Provider**

Get **GPT-5** and **GPT-4** with your API key.

## **Setup**

1. **Get API key** → https://platform.openai.com/api-keys
2. **Add to Polydev** → https://polydev.ai/dashboard → Settings → API Keys → OpenAI
3. **Done** - Polydev uses GPT automatically

## **Models Available**

- **GPT-5** - Best reasoning and complex analysis
- **GPT-4** - Strong general-purpose model

That's it.
