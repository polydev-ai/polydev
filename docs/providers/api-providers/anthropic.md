# **Anthropic Provider**

Get **Claude 4.1 Opus** and **Claude 4 Sonnet** with your API key.

## **Setup**

1. **Get API key** → https://console.anthropic.com/
2. **Add to Polydev** → https://polydev.ai/dashboard → Settings → API Keys → Anthropic
3. **Done** - Polydev uses Claude automatically

## **Models Available**

- **Claude 4.1 Opus** - Best reasoning and complex tasks
- **Claude 4 Sonnet** - Fast and capable for most tasks

That's it.
