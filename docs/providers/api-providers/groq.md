# **Groq Provider**

Get **Grok 4** ultra-fast inference.

## **Setup**

1. **Get API key** → https://console.groq.com/keys
2. **Add to Polydev** → Dashboard → Settings → API Keys → Groq
3. **Done** - Polydev uses Grok automatically

## **Models Available**

- **Grok 4** - Ultra-fast inference for quick iterations

That's it.
