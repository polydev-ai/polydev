# **Providers**

Add your API keys to use AI models.

## **API Providers**

- **[Anthropic](api-providers/anthropic.md)** - Claude 4.1 Opus, Claude 4 Sonnet
- **[OpenAI](api-providers/openai.md)** - GPT-5, GPT-4
- **[Google AI](api-providers/google-ai.md)** - Gemini 2.5 Pro
- **[Groq](api-providers/groq.md)** - Grok 4 (ultra-fast)

## **Setup**

1. **Get API keys** from provider websites
2. **Add to Polydev** → Dashboard → Settings → API Keys
3. **Done** - Polydev routes automatically

That's it.
